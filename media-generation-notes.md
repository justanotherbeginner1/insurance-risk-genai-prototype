# Media Generation with Vertex AI

## Overview

Explored text-to-image generation using Imagen within Vertex AI Studio.

---

## Capabilities Observed

- Photorealistic image generation
- Image refinement (inpainting, outpainting)
- Multiple variations from a single prompt

---

## Responsible AI

- SynthID watermarking enables identification of AI-generated images
- Supports transparency in generative media workflows
