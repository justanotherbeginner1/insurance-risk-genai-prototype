# Insurance Claim Data Extraction

## Objective

Extract structured claim information from unstructured insurance
notifications for faster claims processing.

---

## System Instructions

You are an AI assistant specializing in extracting specific data points
from unstructured insurance claim notifications.

If information is missing, respond with "Not found".
Output each field in a key: value format on a new line.

---

## Extracted Fields

- Policy Number
- Claimant Name
- Date of Loss
- Time of Loss
- Type of Loss
- Description of Damage
- Estimated Loss Amount
- Injuries Reported

---

## Prompting Approach

- Zero-shot prompting for baseline extraction
- Few-shot prompting to improve accuracy and formatting consistency
